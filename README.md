## Enfasis de Texto

* Para nosotros utilizar un texto en **negrita** se utilizan dos "**".
* Para nosotros utilizar un texto en *cursiva* se utiliza un "*".
* Para nosotros ~~tachar un texto~~ necesitas usar dos "~".
